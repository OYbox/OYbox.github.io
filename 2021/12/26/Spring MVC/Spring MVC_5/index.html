    渲染的是 HTML 中的 ，通过 path 与业务数据的属性名对应，

  19. checkbox 标签

    渲染的是 HTML 中的 ，通过 path 与业务数据的属性名对应，可以绑定boolean、数组和集合。
    如果绑定 boolean 类型的变量，该变量值为 true，则表示选中，false 表示不选中。

    如果绑定数组和集合类型，集合中的元素等于 checkbox 的 value 值，则该项选中，否则不选中。

  20. checkboxs 标签

    渲染的是 HTML 中的⼀组 ，这⾥需要结合 items 和 path 两个属性来使⽤，items 绑定被遍历的集合或数组，path 绑定选中的集合或数组，items 是全部选型，path 为默认选中的选型。

    需要注意的是 path 可以直接绑定业务数据的属性，items 则需要通过 EL 表达式从域对象中取值，不能直接写属性名。

  21. radiobutton 标签

    渲染的是 HTML 中的⼀个 ，绑定的数据与标签的 value 值相等为选中状
    态，否则为不选中状态。
