MyBatis Mapper详解

Mapper.xml

  • ⭐statement标签：select、update、delete、insert分别对应查询、更新、删除、插入操作
  • ⭐parameterType：参数数据类型
  1. 基本数据类型，通过id查询Account
  2. String类型，通过name查询Account
  3. 包类型，通过id查询Account
  4. 多个参数，通过name和age查询Account
    可能会报以下错误： 原因是输入参数的时候不可以用#{username}和#{age} 应该用arg1, arg0, param1, param2
  5. Java Bean
  • ⭐resultType：结果类型
  1. 基本数据类型，统计Account总数
  2. 包装类，统计Accoun总数
  3. String类型，通过id查询Account的name
  4. Java bean
